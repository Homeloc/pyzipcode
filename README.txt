Introduction
============

This is a fork from https://github.com/fdintino/pyzipcode

The zipcodes data was found here: http://www.bibichette.com/base-de-donnees-des-villes-francaise/
(Thanks Bibichette!)

Usage examples:

	>>> zcdb = ZipCodeDatabaseFR()
	>>> zcdb.find_ads(72240)
	(u'PAYS DE LA LOIRE', u'Sarthe')
